To run from command line:

java -jar "PaulaPonce.jar" path inputFileName outputFileName

ubuntu example:
java -jar "PaulaPonce.jar" "/home/novakorp/sortLargeFile-main/" fileNumbers2 fileOutSorted
windows example:
java -jar "PaulaPonce.jar" "C:\\prueba\\" fileNumbers2 fileOutSorted
